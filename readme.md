## ˚₊‧꒰ა ɪɴғᴏ ʙᴏᴛ ໒꒱ ‧₊˚

Delta es un bot multifuncional para WhatsApp diseñado para ofrecer entretenimiento, gestión de grupos y herramientas avanzadas con una interfaz elegante ꕤ

* **Creadora:** [Nexy](https://github.com/NexyLove) ✰
* **Node.js + Baileys**
* **Prefijo:** `7`

---

## ˚₊‧꒰ა ɪɴsᴛᴀʟᴀᴄɪóɴ ໒꒱ ‧₊˚

> Sigue estos pasos para tener tu propio Delta-Bot funcionando ꕤ

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/NexyLove/Delta-WaBot.git
    cd Delta-WaBot
    ```

2.  **Instalar dependencias:**
    ```bash
    npm install
    ```

3.  **Configurar credenciales:**
    ```bash
    cp config.js
    ```

4.  **Iniciar el bot:**
    ```bash
    node index.js
    ```
